1. Is there a difference between the field estimated soil moisture and the SLU soil moisture map?¶

  • Field estimated soil moisture = "Soil_class" in DataFrame
  • SLU soil moisture map = "Classified" in DataFrame

"Soil_class" needs to be reclassified to three levels rather than five to match the number of levels of "Classified":

Soil_class -> Soil_class_NEW:

  • Soil_class -> Soil_class_NEW
  • 1 -> 1
  • 2, 3, 4 -> 2
  • 5 -> 3

2. Is soil bearing capacity significantly different in the different soil classes?¶

For the Kruskal-Wallis-Test:

1) create a new dataframe with the columns Classified and Kpa15

2) create dry / mesic / wet

  • dry: sites where "Classified" = 1
  • mesic: sites where "Classified" = 2
  • wet: sites where "Classified" = 3

3) create drypen / mesicpen / wetpen

  • drypen: values of Kpa15 in sites that are classified as dry
  • mesicpen: values of Kpa15 in sites that are classified as mesic
  • wetpen: values of Kpa15 in sites that are classified as wet
In [7]:
classes = df[['Classified', 'Kpa15']] # see 1)

dry = classes[classes['Classified'] == 1] # see 2)
mesic = classes[classes['Classified'] == 2] # see 2)
wet = classes[classes['Classified'] == 3] # see 2)

drypen = dry['Kpa15'] # see 3)
mesicpen = mesic['Kpa15'] # see 3)
wetpen = wet['Kpa15'] # see 3

stats.kruskal(drypen, mesicpen, wetpen) # conduct Kruskal-Wallis test
# < 0.05 > significant difference
# the soil bearing capacity is significantly different between classes

liste_north = []

# choose which locations/y-coordinates count as north/middle/south by looking at the previous graph
for i in range(len(df)):
  if df['Northing'][i] > 6.3e+06 and df['Northing'][i] < 6.4e+06: 
    liste_south.append(df['Northing'][i])
  if df['Northing'][i] > 6.4e+06 and df['Northing'][i] < 7.0e+06: 
    liste_middle.append(df['Northing'][i])
  if df['Northing'][i] > 7.0e+06: 
    liste_north.append(df['Northing'][i])

averagelocation_middle = sum(liste_middle)/len(liste_middle)
averagelocation_north = sum(liste_north)/len(liste_north)
